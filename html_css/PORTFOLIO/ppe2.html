PPE deuxième année : projet MarieTeam

Contexte


Pour ce projet, il nous est demandé de réaliser plusieurs situations professionnels pour l'entreprise fictive MarieTeam, compagnie de transports maritimes.
L'activité de la compagnie se décompose en deux pôles :
- le transport de voyageurs
- le fret, qui correspond au transport de marchandises.


Réalisation


Situation 1 : Application web de gestion de transport voyageur.

Situation 2 : Création d'un client lourd pour l'édition de brochures

Situation 3 : Base de données sur smartphone pour les capitaines

Situation transversale : GLPI


Description


Situation 1 : Création d'une application web permettant au client de MarieTeam la :
- Consultation des liaisons
- Consultation des tarifs
- Consultation des horaires des traversées
- Réservation en ligne avec prise en compte de la fidélisation.

Situation 2 : S'occuper de l’automatisation de l'édition d'une brochure au format PDF, destinée aux clients, présentant les caractéristiques des bateaux réservés aux voyageurs.

Situation 3 : Choisir un modèle de smartphone adapté, donner le schéma de la BDD SQLite locale qu'un technicien devra utiliser pour sauvegarder les éléments de son intervention et le code permettant la sauvegarde, synchroniser les sauvegardes locales des techniciens et les données sur le serveur de BDD, consulter via son Smartphone la BDD sur le serveur

Situation transversale : Mise en place d'une solution de logiciel GLPI.
