Contexte


Plusieurs missions ont été donné afin de valider des compétences du portefeuille.


Réalisations


Mission 3 : Appel d'offre

Mission 4 : Notion de base sur excel

Mission 5 : Projet site web


Description


Mission 3 : Nous devions nous mettre dans la peau d'une entreprise qui repond a un appel d'offre pour equiper informatiquement une entreprise. Le groupe avec le meilleur rapport qualité/prix remportait l'appel d'offre.

Mission 4 : Suite d'exercice permettant de prendre en main le logiciel Excel.

Mission 5 : Créer un site web pour une association sportive ou culturelle de votre choix. Avec mon groupe, nous avons choisi de créer un onglet sur le site web existant d'une association informatique avec l'aide d'un CMS qui decrit un ordinateur et ses composant.

- A1.1.2 , Étude de l'impact de l'intégration d'un service sur le système informatique
